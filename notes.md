# Notes

## File System Management

> Mark when implemented

### Create directory

Command: `mkdir`

Flags: `-p`

### Remove directory

Command: `rmdir`

Flags: `-r`

### Copy

Command: `cp`

### Move/Rename

Command: `mv`

### List directory contents

Command: `ls`

Flags: `-s PROP` - sort by property, `-d` - show details (properties)

### File/Directory Properties

Command: `props`

Flag: `-f pathTOFileOrFolder`

### Download

Command: `download`

Flag: `-f pathToFile`

### Upload

Command: `upload`

Flag: `-f pathToFile`

### Change Directory (Path exists test)

Command: `cd`
