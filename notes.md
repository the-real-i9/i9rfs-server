# Notes

## File System Management

### Copy

Command: `cp`

Arg: `fileOrFolderName`

### Move/Rename

Command: `mv`

Arg: `SOURCE DEST`

### List directory contents

Command: `ls`

Flags: `-s PROP` - sort by property, `-d` - show details (properties)

Arg: `[fileOrFolderName]`

### File/Directory Properties

Command: `props`

Arg: `pathToFileOrFolder`

### Download

Command: `download`

Arg: `pathToFile`

### Upload

Command: `upload`

Arg: `pathToFile`
